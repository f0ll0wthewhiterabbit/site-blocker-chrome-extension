# Easy Block

A Chrome browser extension for blocking sites. Apart from the extension there is a UI that includes authorization and expanded blocking sites functionality.

## Core technologies

### Server

- NestJS
- PostgreSQL
- Prisma
- Docker
- Swagger
- TypeScript

### Client

- Next.js
- React Query
- Tailwind CSS
- Orval
- React Hook Form
- Feature-Sliced Design
- TypeScript

### Extension

- Vite
- CRXJS
- Chrome API
- React Query
- Tailwind CSS
- Feature-Sliced Design
- TypeScript
